re:member

re:member is a memory game with two modes: matching colors and matching sounds. As one of my earliest iOS apps, my focus with this project was to design a simple game with a beautiful user interface. I wanted to create an app that made the user not only enjoy the game, but also just the experience of interacting with the app. Finding a matching pair of tiles reveals a colorful particle animation underneath. Navigating between menu screens within re:member triggers a smooth, sliding animations which define a clear spatial relationship between the views.
